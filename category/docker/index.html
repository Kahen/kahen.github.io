2. 构建可运行的 Jar

我们将使用 Maven 来构建一个可运行的 jar。

因此，我们的应用程序有一个简单的类，HelloWorld.java，其中包含一个 main 方法：

public class HelloWorld {
    public static void main(String[] args){
        System.out.println("Welcome to our application");
    }
}
