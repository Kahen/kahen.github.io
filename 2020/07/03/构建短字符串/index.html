构建短字符串
Created 2020-07-03| Updated 2020-07-03
Post View:

序号：#11
难度：一般
时间限制：1000ms
内存限制：10M
描述
给定任意一个较短的子串，和另一个较长的字符串，判断短的字符串是否能够由长字符串中的字符组合出来，且长串中的每个字符只能用一次。

输入

一行数据包括一个较短的字符串和一个较长的字符串，用一个空格分隔，
如： ab aab bb abc aa cccc uak areuok

输出

如果短的字符串可以由长字符串中的字符组合出来，返回字符串 “true”，否则返回字符串 “false”，注意返回字符串类型而不是布尔型。

输入样例

a b
aa ab
aa aab
uak areuok

输出样例

false
false
true
true

解析：
题目简单，方法简单，但肯定不是最优解哈哈
