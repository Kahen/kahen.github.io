小米OJ上分日志——（三）大数相减

Posted on 2020-07-03

序号：#3
难度：有挑战
时间限制：1000ms
内存限制：100M

描述

两个长度超出常规整形变量上限的大数相减，请避免使用各语言内置大数处理库，如 Java.math.BigInteger 等。

输入
有 N 行测试数据，每一行有两个代表整数的字符串 a 和 b，长度超过百位。规定 a>=b，a, b > 0。 测试结果可以用 linux 小工具 bc进行测试是否正确。

输出
返回表示结果整数的字符串。

输出样例

分析：

该题不能使用内置大数处理库，那么我们可以简单的使用字符数组储存输入字符串，考虑到位数不一致，我们将字符串倒转相减就可以得到结果
