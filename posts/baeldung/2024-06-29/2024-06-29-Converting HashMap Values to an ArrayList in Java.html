2.3 使用Stream API

Java 8带来了许多新特性和增强功能。在这些特性中，我们发现了Stream API。

那么，让我们通过一个实际的例子来说明如何使用流API将HashMap转换为ArrayList：

ArrayList```````````````<String>``````````````` convertUsingStreamApi(HashMap```````<Integer, String>``````` hashMap) {
